What is a Mobile Application?

According to Technopedia a mobile application or an App is a type of software designed to run on a mobile device i.e. a smart phone or tablet. It also states Apple inc as being the company that popularised the type of software and tells the reader that apps are generally smaller individual software units with limited functionality.


Readwrite.com then tells me there are three different categorise mobile applications can fit into.

  • Native Apps
  • Hybrid Apps
  • Web Apps

Native Apps
These are specified applications built for a particular use and they therefore cannot be used on different operating systems. An example of this type of application would be snapchat.


Hybrid Apps
These are a combination of both native and web apps. These applications can be supported across platforms. Think Evernote its beautifully designed and very functional.


Web-based Apps
These are applications that need good internet connection to run and require minimum storage compared to native or hybrid apps. Think of Facebook as an example of this.
